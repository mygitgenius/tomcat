升级指南
    EasyJForum 升级步骤一般如下：

  • 运行程序包中的 install/script/upgrade 目录下的相应版本的数据库升级脚本。
    如果是 Hsqldb 数据库，请自行下载 Hsqldb 管理工具进行操作，在运行此管理工具时可能需要暂停论坛服务。

  • 使用新的程序包中的文件代替旧的程序文件，但 WEB-INF/conf, WEB-INF/data 目录除外。
    如果您自行定制了界面风格，即不通过后台管理而直接对 styles 目录下的文件进行了改动，请先备份这些文件以免被覆盖。

  • 如果 WEB-INF/conf, WEB-INF/data 中的文件需要修改时，会根据不同版本的具体情况另行说明。

  • 删除 WEB-INF/lib 目录下旧版本的 ejforum-*.*.jar 文件。

  • 重新启动论坛服务程序。
